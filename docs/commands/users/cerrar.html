Comando << cerrar >>

Cierra / elimina el canal de voz del anfitrión.
Solo el anfitrión del canal podrá ejecutar este comando
siempre y cuando la configuración del servidor lo permita.
Al ejecutar este comando el canal será eliminado inmediatamente.

Uso del Comando

[prefijo]cerar
